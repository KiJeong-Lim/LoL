# LoL : Lots of Logic

- This repository is my proof pool.

- My environment is `WSL: Ubuntu-20.04`, `Windows 10`.

## Quick Start

```
eval `opam env`
coq_makefile -f _CoqProject -o Makefile
make -j4 -k
```
